## UiLover app

## Pre-requisites
* Android API level 21(Lollipop) and above
* Minimum Sdk Version 21

## Testing App

This is a UiLover app with login and logout functionalities.

## Installation Guidelines
To install the app from an APK file (unknown source):
1. Download the APK file to your device.
2. Enable installation from unknown sources:
    - Open **Settings** on your device.
    - Go to **Security** or **Apps & Notifications** (varies by device).
    - Enable **Install unknown apps** or **Unknown sources** for your preferred browser or file manager.
3. Locate the downloaded APK file (usually in the **Downloads** folder).
4. Tap on the APK file and follow the on-screen instructions to install the app.

> **Note:** Make sure to disable "Install unknown apps" or "Unknown sources" after the app installation for security reasons.

## Features
- Login with username and password.
- Navigate to the homepage after a successful login.
- Logout by clicking the user icon on the home screen.

2. Click the **Login** button to navigate to the homepage.
